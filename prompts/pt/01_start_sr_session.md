# Iniciar uma sessão SR governada

Você está trabalhando em um repositório equipado com a Aurora SR Method.

Objetivo: retomar o projeto corretamente antes de qualquer desenvolvimento.

Instruções:

1. Ler `AGENTS.md`.
2. Ler `docs/codex/SR_BOOTSTRAP.md`.
3. Ler `docs/CURRENT_STATE.md` se existir.
4. Executar `python3 scripts/codex/find_next_session_prompt.py --root . --json` se disponível.
5. Se um `NEXT_SESSION_PROMPT.md` for encontrado, ler antes de propor trabalho.
6. Verificar a versão SR com `python3 scripts/codex/audit_codex_pack.py --root . --json` se disponível.
7. Resumir lots abertos, decisões ativas, bloqueios e próxima ação recomendada.

Não codifique antes de concluir este bootstrap.
