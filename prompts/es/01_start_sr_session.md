# Iniciar una sesión SR gobernada

Estás trabajando en un repositorio equipado con la Aurora SR Method.

Objetivo: retomar el proyecto limpiamente antes de cualquier desarrollo.

Instrucciones:

1. Leer `AGENTS.md`.
2. Leer `docs/codex/SR_BOOTSTRAP.md`.
3. Leer `docs/CURRENT_STATE.md` si existe.
4. Ejecutar `python3 scripts/codex/find_next_session_prompt.py --root . --json` si está disponible.
5. Si se encuentra `NEXT_SESSION_PROMPT.md`, leerlo antes de proponer trabajo.
6. Verificar la versión SR con `python3 scripts/codex/audit_codex_pack.py --root . --json` si está disponible.
7. Resumir lots abiertos, decisiones activas, bloqueos y próxima acción recomendada.

No codifiques antes de completar este bootstrap.
